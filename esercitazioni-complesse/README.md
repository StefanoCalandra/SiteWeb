# Esercitazioni Complesse per `SiteWeb.Api`

Questa cartella è **separata** da `src/SiteWeb.Api` e contiene una raccolta di esercitazioni avanzate pensate per farti lavorare su architettura, scalabilità, sicurezza, qualità del codice e osservabilità.

## Struttura

- `01-architettura-multi-tenant.md`
- `02-cqrs-outbox-event-driven.md`
- `03-hardening-autenticazione-autorizzazione.md`
- `04-grafql-federation-caching.md`
- `05-observability-sre-chaos.md`
- `06-performance-benchmarking-capacity.md`
- `07-data-governance-migrazioni-zero-downtime.md`
- `08-reliability-resilience-patterns.md`

## Come usarle

1. Scegli una esercitazione.
2. Crea un branch dedicato.
3. Definisci backlog tecnico (epic > story > task).
4. Implementa con test, metriche e documentazione.
5. Fai una retro tecnica finale con decision log (ADR).

Ogni esercitazione include:
- contesto;
- obiettivi;
- requisiti vincolanti;
- criteri di accettazione;
- estensioni opzionali.
